# 🐍 Estrutura de Dados com Python | Guilheme Nixon

Os exercícios são selecionados de atividades acadêmicas(UFMG, UFRJ, UnB, CEFET), processos seletivos, provas e projetos reais.

## Tecnologias

- Python 3.10+
- Sem bibliotecas externas — todas as estruturas implementadas do zero
- Os níveis de dificuldade são classificações pessoais
- O projeto está em constante atualização, se houver algum exercício sem resolução está em andamento
